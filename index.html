# URBANOpt REopt Gem

# Rdocs

The URBANopt REopt Gem extends the URBANopt Scenario Gem to provide functionality for running a REopt Lite optimization for Scenario Reports and Feature Reports.

REopt Lite is techno-economic decision support model, maintained by the National Renewable Energy Laboratory (NREL), that is publicly accessible via API with an API key obtained from NREL's Developer Network. REopt Lite uses mixed-integer linear programming to identify deterministic distributed energy resource (DER) solutions that minimize life cycle energy costs to a site. From set of possible generation technologies (i.e. Solar PV, Storage, Wind, Conventional Generator), the model selects the optimal technology capacities and dispatch strategies. Problems formulated through this framework are subject to customizable set of costs (i.e. capital, operations and maintenance, fuel), constraints (i.e. minimum and maximum capacity) and financial assumptions (i.e. discount factor, escalation rates, net metering policy).

The URBANopt Example Project demonstrates how to use the URBANopt REopt Gem to perform a scenario analysis.

# Installation

Add this line to your application's Gemfile:

gem 'urbanopt-reopt'

And then execute:

$ bundle install
$ bundle update

Or install it yourself as:

$ gem install 'urbanopt-reopt'

# Testing

Check out the repository and then execute:

$ bundle install
$ bundle update    
$ bundle exec rake

# Releasing

  • Update change log
  • Update version in /lib/urbanopt/reopt/version.rb
  • Merge down to master
  • run rake release from master
